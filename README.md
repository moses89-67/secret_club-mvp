# SECRET CLUB® — MVP Website

A minimal, conversion-focused landing page for **SECRET CLUB®**, a streetwear brand built around exclusivity, identity, and limited drops.

---

## 🧠 Concept

SECRET CLUB® is not just clothing — it is access.

This MVP is designed to:
- Capture emails (waitlist for drops)
- Drive Instagram followers (main growth channel)
- Present brand identity in a clean, fast way
- Prepare for future e-commerce expansion

---

## 🚀 Features

### ✔ Hero Section
Strong brand identity + clear positioning

### ✔ Email Waitlist
Simple lead capture form for upcoming drops

### ✔ Collections Preview
Minimal product storytelling (no store complexity yet)

### ✔ Instagram Funnel
Primary conversion channel (@secretclub.cl)
